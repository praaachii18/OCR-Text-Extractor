# assets/

Place sample images here for testing the OCR extractor.

## Suggested test images

- `sample_invoice.png` — a printed invoice or receipt
- `sample_handwriting.jpg` — handwritten text
- `sample_document.pdf` — a multi-page PDF
- `sample_low_quality.jpg` — a blurry/noisy photo of text

## Where to get test images

- **Google Images** — search for "invoice sample PNG"
- **Your scanner / phone camera** — photograph any printed text
- **PDF samples** — any text-heavy PDF document
